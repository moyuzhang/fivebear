| 开发语言     | Java               | 17     | 企业级应用生态支持            |

## 3. 核心组件说明
### 认证服务架构
```mermaid
sequenceDiagram
    用户->>+认证服务: 登录请求
    认证服务->>LDAP: 身份验证
    LDAP-->>认证服务: 验证结果
    认证服务->>JWT: 生成令牌
    认证服务-->>用户: 返回访问令牌
```
